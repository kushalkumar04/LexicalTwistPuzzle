# Lexical Twist Puzzle

A console-based Java application demonstrating progressive OOPS design and GitFlow-based development.

## Features

- Input validation (single word constraint)
- Reverse relationship detection
- Vowel transformation logic
- Word merging and vowel–consonant distribution analysis
- Rule-based unique character extraction
- Fully modular architecture

## Architecture

- LexicalTwistPuzzle → Input/Output
- WordValidator → Validation logic
- LexicalAnalyzer → Core puzzle logic

## OOPS Principles Demonstrated

- Encapsulation
- Abstraction
- Single Responsibility Principle
- Modularity
- Maintainability
